# 🍋 Lemoni To Do List

## Lemoni To Do List란?

---

간단하게 할 일을 저장하고, 관리할 수 있는 프로그램입니다.

-> [Demo](https://lemoni-todo.netlify.app/)

<br/>

## How to start

---

```shell
npm install

npm start
```

<br/>

## 주요 기능

---

- 할 일 생성, 수정, 삭제
- 완료된 할 일 표시(체크)

## 사용 기술

---

`React`, `TypeScript`, `styled-components`, `recoil`, `fontawesome`

<br/>

## 추가 예정 항목

---

[ ] 여백 입력 후 할 일 저장 시 모달<br/>
[ ] 삭제 버튼 클릭 시 확인 모달<br/>
[ ] 날짜 별 할 일 내역 보기
